Heuristic solution method:

To run the heuristic solution method make sure you have Python/11 and Maven/3.6.3 installed.

Parameters can be changed in the HackInitProblemData.java file

To run a series of instances, this can be done in the Main.java file. Scroll down to get_task_list and in the scenario_numbers array enter the scenario numbers you want to run.

The Maven project is run by going into the KodeMaster directory (where the pom file is located).

Compile the project by typing 'mvn clean package' - this creates a folder named target with a runnable .jar file

change directory to the target folder and run the code by typing 'java -jar KodeMaster-1.0-SNAPSHOT.jar'
