Exact solution method

To run the exact solution method make sure you have Python/3.7.4 installed with Gurobi/9.0.2

To run the model enter 'main.py' and in afm.solve('#') Replace the # with the instance you want to run, then run the main method as a normal Python script.

To change parameeters such as weather scenario or discrete time periods per hour, this can be done in the 'data.py' file.
